Compatible with any 3D file

270 Degrees can work with any 3D compatible file including CAD, CLO, GLB, and STL. Assets are hosted on the API to ensure optimal file size and computational load so great performance is guaranteed. What's more the API detects the end user's device's capabilities and automatically delivers a suitable asset for the best experience.
